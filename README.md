# pMSE — Reporte de las 10 redes (v0)

Examen mental por redes neurales. Una ficha por red (del tronco encefálico al ápex
integrador) con tipo ontológico, evidencia, mapeo fenómeno→sustrato→fuente, escala de
puntuación −2/+2 y fortaleza de la evidencia.
